Friv is an online gaming website that offers a wide variety of free, browser-based games across multiple genres, including puzzle, adventure, action, and arcade. It is designed with a minimalist interface, allowing users to easily browse and play games without the need for downloads or sign-ups. The platform is especially popular among children and casual gamers due to its simple navigation and colorful design. Friv’s games are typically lightweight and focus on quick, entertaining experiences rather than complex gameplay. Its extensive library caters to diverse interests, making it a go-to site for quick and fun gaming sessions.

The overall HTML structure of the website follows conventions usually used in creating websites. Starting with <!DOCTYPE html> to inform the browser this is an HTML file. Next, the <html> with the <head> and <body> tags making up the skeleton of the website. The <head> section contains information for the document through defining rules the file follows such as the title tag which is what appears on the browser's tab and <meta> tags which specifies the character set, provides a description for the site. The <head> tag also imports information and assets from other sources to be used by the HTML file. This is done via the various <link> and <script> tags which provide links to the sources of assets the site uses or scripts that the site will execute. The <body> of the website will be explained in depth in the next section.

The <header> is nowhere to be found, instead the <div class="headerSearchResult"> and the <div id="logoContainer"> can be found, featuring the logo and the search button respectively. The footer isn’t stored in a <footer> element but a <div class="footerSection">, similar to the missing <header>. The footerSection starts off with a spacer to separate the section above it from the footer. Next is a <div> for feedback which is currently hidden. Below that is a <div class="hueBlock"> used as a spacer to visually differentiate the space above with the <div> for "copyright" which contains the copyright information of friv.com. Then another <div class="hueBlock"> ends the website. Lastly, numerous scripts and hidden divs are placed at the bottom of the file. Overall the HTML structure mostly follows conventions such as following the basic HTML skeleton, proper nesting, and correct indentation, the website could improve with better use of semantic HTML elements. The inclusion of a proper <header> and <footer> element would make the website more organized, improving structure and readability.

Game Display

Fig. 1.5: HTML Games Display

As for the main content of the website, the Games display, it is entirely stored within <main id="gamesBlock">. Within that section of the website are 296 versions of <div class="gameThumbContainer"> which are used to display the main content of the website, friv.com. This allows the user to select which game they would like to select and play. This is done through the elements contained in <div class="gameThumbContainer">, which are: <div class="gameThumbShine"> containing a shine style element for the game thumbnail; an <img> element of the game's thumbnail within an <a> anchor link to direct the user to the game; and <div class="gameTitle"> which contains the game's title. Overall, this is an efficient way of showcasing the games the website has to offer, allowing style customizability options that are convenient to use and quick to change.

Responsiveness Design Techniques

The website has managed to avoid repeated files and inefficient property usage. It ensured the website to run smoothly and faster to deliver its primary purpose to its end users. With the multitude of games it provided, it was still able to log properly without any form of lagging thus contributing to an overall positive user experience.

Performance and Effeciency

The website entails to be specific when it comes to its flow and direction. It utilizes the right selectors to provide its needed qualities, thus ensuring that chaining and assigning of files and properties were made to be fast and efficient.
